# Filter by Field Presence  
<LanguageSwitcher supportedLanguages={supportedLanguages} />
<LanguageContent language="csharp">

<Admonition type="note" title="Note">

* To query for documents that contain a particular field,  
  use extension method `WhereExists()` which is accessible from [DocumentQuery](../../../client-api/session/querying/document-query/what-is-document-query).  

* A document that doesn't contain the specified field will be excluded from the query results.

* In this page:  
  * [Filter by field name](../../../client-api/session/querying/how-to-filter-by-field#filter-by-field-name)  
  * [Filter by field path](../../../client-api/session/querying/how-to-filter-by-field#filter-by-field-path)  
  * [Syntax](../../../client-api/session/querying/how-to-filter-by-field#syntax)

</Admonition>
## Filter by field name 

<Tabs groupId='languageSyntax'>
<TabItem value="Sync" label="Sync">
<CodeBlock language="csharp">
{`// Only documents that contain field 'FirstName' will be returned

List<Employee> results = session
    .Advanced
    .DocumentQuery<Employee>()
    .WhereExists("FirstName")
     // Or use lambda expression: .WhereExists(x => x.FirstName)
    .ToList();
`}
</CodeBlock>
</TabItem>
<TabItem value="Async" label="Async">
<CodeBlock language="csharp">
{`// Only documents that contain field 'FirstName' will be returned

List<Employee> results = await asyncSession
    .Advanced
    .AsyncDocumentQuery<Employee>()
    .WhereExists("FirstName")
     // Or use lambda expression: .WhereExists(x => x.FirstName)
    .ToListAsync();
`}
</CodeBlock>
</TabItem>
<TabItem value="RQL" label="RQL">
<CodeBlock language="sql">
{`// Only documents that contain field 'FirstName' will be returned

from Employees
where exists("FirstName")
`}
</CodeBlock>
</TabItem>
</Tabs>

<TabItem value="something" label="whereExists_syntax">
<CodeBlock language="csharp">

IDocumentQuery<T> WhereExists<TValue>(Expression<Func<T, TValue>> propertySelector);
`}
</CodeBlock>
</TabItem>

| Parameters           | Type                         | Description                                                           |
|----------------------|------------------------------|-----------------------------------------------------------------------|
| **fieldName**        | `string`                     | The name / path of the document field to filter by                    |
| **propertySelector** | `Expression<Func<T,TValue>>` | Lambda expression with name / path of the document field to filter by |

## Filter by Field Name or Path

To search for documents that contain a specific field by the field's **name** or **path**, 
pass the name or the path to `where_exists` as demonstrated below.  
### Syntax 

<TabItem value="something-something" label="whereExists_syntax">
<CodeBlock language="python">
{`def where_exists(self, field_name: str) -> DocumentQuery[_T]: ...
`}
</CodeBlock>
</TabItem>

| Parameters           | Type        | Description                               |
|----------------------|-------------|-------------------------------------------|
| **field_name**       | `str`       | Field Name or Path to filter documents by |
### Examples

Pass `where_exists` a string containing the field name or path.  

* **Pass a Field Name**:
<Tabs groupId='languageSyntax'>
<TabItem value="Query" label="Query">
<CodeBlock language="python">
{`# Only documents that contain field 'first_name' will be returned
results = list(session.advanced.document_query(object_type=Employee).where_exists("first_name"))
`}
</CodeBlock>
</TabItem>
<TabItem value="RQL" label="RQL">
<CodeBlock language="sql">
{`// Only documents that contain the 'FirstName' field will be returned

from Employees
where exists("FirstName")
`}
</CodeBlock>
</TabItem>
</Tabs>

* **Pass a Field Path**:
<Tabs groupId='languageSyntax'>
<TabItem value="Query" label="Query">
<CodeBlock language="python">
{`results = list(
    session.advanced.document_query(object_type=Employee).where_exists("address.location.latitude")
)
`}
</CodeBlock>
</TabItem>
<TabItem value="RQL" label="RQL">
<CodeBlock language="sql">
{`// Only documents that contain the 'Latitude' property in the specified path will be returned

from Employees
where exists("Address.Location.Latitude")
`}
</CodeBlock>
</TabItem>
</Tabs>
