# Glossary: Node Tag

### What is a Node Tag?

It is a [cluster node](cluster-node) unique identifier, which is a denoted by up to four upper-case letters.

Once a node is added to [RavenDB cluster](ravendb-cluster), a node tag will be assigned to it.
