## Paging results

* Get and process revisions, one page at a time:
<TabItem value="getRevisionsWithPaging" label="getRevisionsWithPaging">
<CodeBlock language="python">
{`start = 0
page_size = 100

while True:
    # Execute the get revisions operation
    # Pass the document id, start & page size to get
    revisions = store.operations.send(GetRevisionsOperation("comapnies/1-A", Company, start, page_size))

    # Process the retrieved revisions here

    if len(revisions.results) < page_size:
        break  # No more revisions to retrieve

    # Increment 'start' by page-size, to get the "next page" in next iteration
    start += page_size
`}
</CodeBlock>
</TabItem>

* The document ID, start & page size can be wrapped in a `Parameters` object:
<TabItem value="getRevisionsWithPagingParams" label="getRevisionsWithPagingParams">
<CodeBlock language="python">
{`parameters = GetRevisionsOperation.Parameters("companies/1-A", 0, 100)

revisions = store.operations.send(GetRevisionsOperation.from_parameters(parameters))
`}
</CodeBlock>
</TabItem>



## Syntax

| Parameter | Type | Description |
| - | - | - |
| **id** | `str` | ID of the document to get revisions for |
| **start** | `int` | Revision number to start from |
| **page_size** | `int` | Number of revisions to get |
| **parameters** | `Parameters` | An object that wraps `id`, `start`, and `page_size` |
