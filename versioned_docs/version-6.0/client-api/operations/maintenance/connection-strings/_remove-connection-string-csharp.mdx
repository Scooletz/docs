You can remove a connection string by using **RemoveConnectionStringOperation**.

## Syntax

<TabItem value="remove_connection_string" label="remove_connection_string">
<CodeBlock language="csharp">
{`public RemoveConnectionStringOperation(T connectionString)
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ----- | ---- |
| **connectionString** | `T` | Connection string to remove: `RavenConnectionString` or `SqlConnectionString` |

## Example - Remove Raven Connection String

<TabItem value="remove_raven_connection_string" label="remove_raven_connection_string">
<CodeBlock language="csharp">
{`RemoveConnectionStringOperation<RavenConnectionString> operation
    = new RemoveConnectionStringOperation<RavenConnectionString>(
        connectionString);

store.Maintenance.Send(operation);
`}
</CodeBlock>
</TabItem>
