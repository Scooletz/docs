# Plugins: Database configuration

To alter database configuration you can edit the configuration document (more about how it can be done and what configuration options are available can be found [here](../../server/configuration/configuration-options.mdx)) but sometimes it might be better to change configuration programmatically e.g. imagine a situation, where you have 100 databases and you want to change one setting in each of them. This is why the `IAlterConfiguration` interface was created.

<TabItem value="plugins_5_0" label="plugins_5_0">
<CodeBlock language="csharp">
{`public interface IAlterConfiguration
\{
	void AlterConfiguration(InMemoryRavenConfiguration configuration);
\}
`}
</CodeBlock>
</TabItem>

## Example - Disable compression

<TabItem value="plugins_5_1" label="plugins_5_1">
<CodeBlock language="csharp">
{`public class CommonConfiguration : IAlterConfiguration
\{
	public void AlterConfiguration(InMemoryRavenConfiguration configuration)
	\{
		configuration.HttpCompression = false;
	\}
\}
`}
</CodeBlock>
</TabItem>
