# Status: Logs

This view allows browsing through last 500 log entries, filtered by severity. You can also sort them using various search phrases, or by log level, logged time, etc.

![Figure 1. Studio. Querying Static Index.](./assets/status_logs-1.png)

<Admonition type="info" title="Info">
Remember that the view is **not** refreshed automatically and to download most recent logs from the server you need to click the `Refresh` button.
</Admonition>
