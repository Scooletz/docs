# Status: Storage

`On disk` - shows general information about space used and paths to logs, data and indexes.

![Figure 1. Studio. Status. Storage. On disk.](./assets/status-storage-1.png)

`Internal storage breakdown` - shows detailed information about space used. 

![Figure 2. Studio. Status. Storage. Internal storage breakdown.](./assets/status-storage-2.png)
