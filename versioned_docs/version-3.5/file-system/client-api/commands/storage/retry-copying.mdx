#Commands: RetryCopyingAsync

**RetryCopyingAsync** runs a background task that will resume unaccomplished file copies. Read [Background tasks](../../../server/background-tasks.mdx) article for details.

## Syntax

<TabItem value="retry_copying_1" label="retry_copying_1">
<CodeBlock language="csharp">
{`Task RetryCopyingAsync();
`}
</CodeBlock>
</TabItem>


| Return Value | |
| ------------- | ------------- |
| **Task** |  A task that represents the asynchronous operation |
