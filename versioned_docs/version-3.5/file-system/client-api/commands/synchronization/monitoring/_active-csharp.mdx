#Commands: GetActiveAsync

The **GetActiveAsync** method returns the information about the active outgoing synchronizations.



<TabItem value="get_active_1" label="get_active_1">
<CodeBlock language="csharp">
{`Task<ItemsPage<SynchronizationDetails>> GetActiveAsync(int start = 0, int pageSize = 25);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **start** | int | The number of items to skip |
| **pageSize** | int | The maximum number of items to get |

---

| Return Value | |
| ------------- | ------------- |
| **Task&lt;ItemsPage&lt;SynchronizationDetails&gt;&gt;** | A task that represents the asynchronous operation. The task result is an `ItemsPage` object that contains the number of total results and the list of the [`SynchronizationDetails`](../../../../../glossary/synchronization-details.mdx) objects, which contains info about synchronizations being in progress. |
