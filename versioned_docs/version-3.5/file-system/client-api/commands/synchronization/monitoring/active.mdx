#Commands: GetActiveAsync

The **GetActiveAsync** method returns the information about the active outgoing synchronizations.



<TabItem value="get_active_1" label="get_active_1">
<CodeBlock language="csharp">
{`Task<ItemsPage<SynchronizationDetails>> GetActiveAsync(int start = 0, int pageSize = 25);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **start** | int | The number of items to skip |
| **pageSize** | int | The maximum number of items to get |

---

| Return Value | |
| ------------- | ------------- |
| **Task&lt;ItemsPage&lt;SynchronizationDetails&gt;&gt;** | A task that represents the asynchronous operation. The task result is an `ItemsPage` object that contains the number of total results and the list of the [`SynchronizationDetails`](../../../../../glossary/synchronization-details.mdx) objects, which contains info about synchronizations being in progress. |

#Commands: GetActive

The **GET** method returns the information about the active outgoing synchronizations.



<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverUrl\}/fs/\{fileSystemName\}/synchronization/active?start=\{start\}&pageSize=\{pageSize\}  \\
	-X GET
`}
</CodeBlock>
</TabItem>

### Request

| Query parameter | Required | Description |
| ------------- | -- | ---- |
| **start** | No | The number of results that should be skipped |
| **pageSize** | No | The maximum number of results that will be returned |

### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |

| Return Value | Description |
| ------------- | ------------- |
| **JSON** | The `ItemsPage` object that contains the number of total results and the list of the [`SynchronizationDetails`](../../../../../glossary/synchronization-details.mdx) objects, which contains info about synchronizations being in progress. |
