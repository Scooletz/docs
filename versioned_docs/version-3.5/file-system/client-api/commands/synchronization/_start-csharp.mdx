#Commands: StartAsync

The **StartAsync** method is used to manually force the synchronization to the destinations. It has two overloads that allow you either to synchronize all the files which require that or to synchronize just one specified file.



<TabItem value="start_1" label="start_1">
<CodeBlock language="csharp">
{`Task<DestinationSyncResult[]> StartAsync(bool forceSyncingAll = false);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **forceSyncingAll** | bool | Determines whether finished synchronization should schedule a next pending one (there can be only [a limited number of concurrent synchronizations](../../../synchronization/configurations.mdx#ravensynchronizationconfig) to a destination file system). The reports of such synchronizations will *not* be included in `DestinationSyncResult` object. |


---

| Return Value | |
| ------------- | ------------- |
| **Task&lt;DestinationSyncResult[]&gt;** | A task that represents the asynchronous synchronization operation. The task result is an array of `DestinationSyncResult` object that contains reports about performed operations. |

## Synchronization of a particular file to a single file system

<TabItem value="start_2" label="start_2">
<CodeBlock language="csharp">
{`Task<SynchronizationReport> StartAsync(string filename, SynchronizationDestination destination);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **filename** | string | The full file name |
| **destination** | SynchronizationDestination | The destination file system |


---

| Return Value | |
| ------------- | ------------- |
| **Task&lt;SynchronizationReport&gt;** | A task that represents the asynchronous file synchronization operation. The task result is a [`SynchronizationReport`](../../../../glossary/synchronization-report.mdx).  |


### Example

<TabItem value="start_5" label="start_5">
<CodeBlock language="csharp">
{`SynchronizationReport syncReport = await store.AsyncFilesCommands.Synchronization
											.StartAsync("/products/pictures/canon_1.jpg", 
														new SynchronizationDestination
														\{
															FileSystem = "NorthwindFS",
															ServerUrl = "http://localhost:8081/"
														\});
`}
</CodeBlock>
</TabItem>
