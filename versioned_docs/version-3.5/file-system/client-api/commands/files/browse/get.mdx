#Commands: GetAsync

**GetAsync** is used to get the file headers of the selected files.



<TabItem value="get_1" label="get_1">
<CodeBlock language="csharp">
{`Task<FileHeader[]> GetAsync(string[] filenames);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **filenames** | string[] | Names of the files you want to get headers for. |

---

| Return Value | |
| ------------- | ------------- |
| **Task&lt;FileHeader[]&gt;** | A task that represents the asynchronous get operation. The task result is the array of [`FileHeaders`](../../../../../glossary/file-header.mdx). |

## Example

<TabItem value="get_2" label="get_2">
<CodeBlock language="csharp">
{`FileHeader[] icons = await store
	.AsyncFilesCommands
	.GetAsync(new[]
	\{
		"/images/icons/small/1.png", "/images/icons/large/1.png"
	\});
`}
</CodeBlock>
</TabItem>

</LanguageContent>
<LanguageContent language="http">
#Commands: Get

The **GET** method with **fileNames** parameters is used to retrieve headers of multiple files.



<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverUrl\}/fs/\{fileSystemName\}/files?fileNames=\{name1\}&fileNames=\{name2\}...  \\
	-X GET
`}
</CodeBlock>
</TabItem>

### Request

| Query parameter | Required | Description |
| ------------- | -- | ---- |
| **fileNames** | Yes | One or many names of the files you want to get headers for |

### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |

| Return Value | Description |
| ------------- | ------------- |
| **Array** | The response consists of array of [file headers](../../../../../glossary/file-header.mdx) |

---
