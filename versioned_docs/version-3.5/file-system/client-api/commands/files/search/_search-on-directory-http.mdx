#Commands: SearchOnDirectory

In order to look for files located in a given directory use the standard [search method](./search) with `__directoryName` and `__level` [terms](../../../../indexing.mdx)
specified in the query parameter.



Download all files located in `/movies` directory which names end with `.avi`:

<TabItem value="json" label="json">
