#Session: Querying: StreamQueryAsync

Query results can be streamed using **StreamQueryAsync** method from Advanced session operations.



<TabItem value="streaming_1" label="streaming_1">
<CodeBlock language="csharp">
{`Task<IAsyncEnumerator<FileHeader>> StreamQueryAsync(IAsyncFilesQuery<FileHeader> query);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **query** | IAsyncFilesQuery&lt;FileHeader&gt; | Query to stream results for |

---

| Return Value | |
| ------------- | ------------- |
| **Task&lt;IAsyncEnumerator&lt;FileHeader&gt;&gt;** | A task that represents the asynchronous operation. The task result is the enumerator of [`FileHeaders`](../../../../../glossary/file-header.mdx) objects. |

## Example

<TabItem value="streaming_2" label="streaming_2">
