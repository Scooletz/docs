# Commands: How to compact database?
<LanguageSwitcher supportedLanguages={supportedLanguages} />
<LanguageContent language="csharp">


To compact a database, please use `CompactDatabase` command available in `GlobalAdmin`.

## Syntax

<TabItem value="compact_1" label="compact_1">
<CodeBlock language="csharp">
{`Operation CompactDatabase(string databaseName);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **databaseName** | string | Name of a database to compact |

## Example

<TabItem value="compact_2" label="compact_2">
<CodeBlock language="csharp">
{`Operation operation = store
	.DatabaseCommands
	.GlobalAdmin
	.CompactDatabase("Northwind");

operation.WaitForCompletion();
`}
</CodeBlock>
</TabItem>

## Remarks

Compacting operation is executed **asynchronously** and during this operation the **database** will be **offline**.


</LanguageContent>
<LanguageContent language="java">


To compact a database, please use `compactDatabase` command available in `GlobalAdmin`.

## Syntax

<TabItem value="compact_1" label="compact_1">
<CodeBlock language="java">
{`public Operation compactDatabase(String databaseName);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **databaseName** | String | Name of a database to compact |

## Example

<TabItem value="compact_2" label="compact_2">
<CodeBlock language="java">
{`Operation operation = store.getDatabaseCommands().getGlobalAdmin().compactDatabase("Northwind");
operation.waitForCompletion();
`}
</CodeBlock>
</TabItem>

## Remarks

Compacting operation is executed **asynchronously** and during this operation the **database** will be **offline**.


</LanguageContent>
<LanguageContent language="http">


To compact a database, please use compact endpoint.

## Syntax

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverName\}/admin/compact?database=\{databaseName\} \\
	-X POST
`}
</CodeBlock>
</TabItem>

### Request

| Query parameter | Required | Description |
| ------------- | -- | ---- |
| **databaseName** | Yes | Name of a database to compact |

### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |

| Return Value | Description |
| ------------- | ------------- |
| **OperationId** | Operation id |

---

## Example

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl -X POST "http://localhost:8080/admin/compact?database=NorthWind"
< HTTP/1.1 200 OK
\{"OperationId":1\}
`}
</CodeBlock>
</TabItem>

## Remarks

Compacting operation is executed **asynchronously** and during this operation the **database** will be **offline**.


</LanguageContent>
