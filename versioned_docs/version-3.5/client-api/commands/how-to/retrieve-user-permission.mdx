# Commands: How to retrieve user permission for a specified database?

`GetUserPermission` used to retrieve user permissions for a specified database

## Syntax

<TabItem value="get_user_permission_1" label="get_user_permission_1">
<CodeBlock language="csharp">
{`UserPermission GetUserPermission(string database, bool readOnly);
`}
</CodeBlock>
</TabItem>


| Parameters | | |
| ------------- | ------------- | ----- |
| **database** | string | name of the database we want to retrive the permissions |
| **readOnly** | bool | the type of the operations allowed, read only , or read-write |


| Return Value | |
| ------------- | ----- |
| [UserPermission](../../../glossary/user-permission.mdx) | Retrieves user permissions for a specified database. |

## Example I

<TabItem value="get_user_permission_2" label="get_user_permission_2">
<CodeBlock language="csharp">
{`var per = store.DatabaseCommands.GetUserPermission("Foo", false);
var isGrant = per.IsGranted;
var res = per.Reason;
`}
</CodeBlock>
</TabItem>
