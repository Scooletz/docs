Commands are a set of low level operations that can be used to manipulate data and change configuration on a server.

They are available in **[DocumentStore](../../client-api/what-is-a-document-store.mdx)** under **getDatabaseCommands()**.

<TabItem value="what_are_commands_1" label="what_are_commands_1">
<CodeBlock language="java">
{`IDatabaseCommands commands = store.getDatabaseCommands();
`}
</CodeBlock>
</TabItem>
