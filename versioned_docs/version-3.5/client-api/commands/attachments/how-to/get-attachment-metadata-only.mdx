# Attachments: How to get attachment metadata only?
<LanguageSwitcher supportedLanguages={supportedLanguages} />
<LanguageContent language="csharp">


There are few methods that allow you to download attachment metadata from a database:   
- [HeadAttachment](../../../../client-api/commands/attachments/how-to/get-attachment-metadata-only.mdx#head)   
- [GetAttachmentHeadersStartingWith](../../../../client-api/commands/attachments/how-to/get-attachment-metadata-only.mdx#getattachmentheadersstartingwith)   

## **HeadAttachment**

**HeadAttachment** can be used to download attachment metadata for a single attachment.

### Syntax

<TabItem value="head_1_0" label="head_1_0">
<CodeBlock language="csharp">
{`Attachment HeadAttachment(string key);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **key** | string | key of the attachment you want to download metadata for |

| Return Value | |
| ------------- | ----- |
| [Attachment](../../../../glossary/attachment.mdx) | Object that represents attachment. |

### Example

<TabItem value="head_1_1" label="head_1_1">
<CodeBlock language="csharp">
{`Attachment attachment = store
	.DatabaseCommands
	.HeadAttachment("albums/holidays/sea.jpg"); // null if does not exist
`}
</CodeBlock>
</TabItem>



## **GetAttachmentHeadersStartingWith**

**GetAttachmentHeadersStartingWith** can be used to download attachment metadata for a multiple attachments.

### Syntax

<TabItem value="head_2_0" label="head_2_0">
<CodeBlock language="csharp">
{`IEnumerable<Attachment> GetAttachmentHeadersStartingWith(
	string idPrefix,
	int start,
	int pageSize);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **idPrefix** | string | prefix for which attachments should be returned |
| **start** | int | number of attachments that should be skipped |
| **pageSize** | int | maximum number of attachments that will be returned |

| Return Value | |
| ------------- | ----- |
| [AttachmentInformation](../../../../glossary/attachment-information.mdx) | Object that represents attachment metadata information. |

### Example

<TabItem value="head_2_1" label="head_2_1">
<CodeBlock language="csharp">
{`IEnumerable<Attachment> attachments = store
	.DatabaseCommands
	.GetAttachmentHeadersStartingWith("albums/holidays/", 0, 10);
`}
</CodeBlock>
</TabItem>



## Remarks

**Data** property in `Attachment` will return empty stream for the above methods.

## **HeadAttachment**

**HeadAttachment** can be used to download attachment metadata for a single attachment.

### Syntax


<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverName\}/databases/\{databaseName\}/static/\{key\} \\
	-X HEAD
`}
</CodeBlock>
</TabItem>

### Request

| Query parameter | Required | Description |
| ------------- | -- | ---- |
| **key** | Yes | key of the attachment you want to download metadata for |

### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |

| Header | Description |
| -------- | - |
| any header | attachment metadata |

---

### Example

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl -X HEAD "http://localhost:8080/databases/sample/static/sea.jpg"
< HTTP/1.1 200 OK
< ETag: "02000000-0000-0002-0000-000000000002"
< Description: Holidays 2014
`}
</CodeBlock>
</TabItem>



## **GetAttachmentHeadersStartingWith**

**GetAttachmentHeadersStartingWith** can be used to download attachment metadata for a multiple attachments.

### Syntax

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverName\}/databases/\{databaseName\}/static? \\
		&startsWith=\{startsWith\}  \\
		&start=\{start\} \\
		&pageSize=\{pageSize\} \\
	-X GET
`}
</CodeBlock>
</TabItem>

### Request

| Query parameter | Required | Description |
| ------------- | -- | ---- |
| **idPrefix** | Yes | prefix for which attachments should be returned |
| **start** | No | number of attachments that should be skipped |
| **pageSize** | No | maximum number of attachments that will be returned |

### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |

| Return Value | Description |
| ------------- | ------------- |
| payload | list of [AttachmentInformation](../../../../glossary/attachment-information.mdx) |

---

### Example


<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl -X GET "http://localhost:8080/databases/sample/static/?startsWith=albums/&pageSize=128&start=0"
< HTTP/1.1 200 OK
[
	\{
		"Size":4,
		"Key":"sea.jpg",
		"Metadata":\{"Description":"Holidays 2014","Content-Type":"application/json; charset=UTF-8"\},
		"Etag":"02000000-0000-0002-0000-000000000001"
	\},
	... another attachment information ...
]
`}
</CodeBlock>
</TabItem>
