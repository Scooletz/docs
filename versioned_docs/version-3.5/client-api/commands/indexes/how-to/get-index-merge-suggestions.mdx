# Commands: Indexes: How to get index merge suggestions?
<LanguageSwitcher supportedLanguages={supportedLanguages} />
<LanguageContent language="csharp">


**GetIndexMergeSuggestions** will retrieve all suggestions for an index merging.

## Syntax

<TabItem value="merge_suggestions_1" label="merge_suggestions_1">
<CodeBlock language="csharp">
{`IndexMergeResults GetIndexMergeSuggestions();
`}
</CodeBlock>
</TabItem>

| Return Value | |
| ------------- | ----- |
| [IndexMergeResults](../../../../glossary/index-merge-results.mdx) | Merge suggestions for indexes. |

## Example

<TabItem value="merge_suggestions_2" label="merge_suggestions_2">
<CodeBlock language="csharp">
{`IndexMergeResults suggestions = store.DatabaseCommands.GetIndexMergeSuggestions();
`}
</CodeBlock>
</TabItem>


</LanguageContent>

<LanguageContent language="http">


**GetIndexMergeSuggestions** will retrieve all suggestions for an index merging.

### Syntax

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverName\}/databases/\{databaseName\}/debug/suggest-index-merge \\
	-X GET 
`}
</CodeBlock>
</TabItem>

### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |

| Return Value | Description |
| ------------- | ------------- |
| payload | [IndexMergeResults](../../../../glossary/index-merge-results.mdx) |
