# Attachments: Delete
<LanguageSwitcher supportedLanguages={supportedLanguages} />
<LanguageContent language="csharp">


**DeleteAttachment** is used to remove an attachment from a database.

## Syntax

<TabItem value="delete_1" label="delete_1">
<CodeBlock language="csharp">
{`void DeleteAttachment(string key, Etag etag);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **key** | string | key of an attachment to delete |
| **etag** | Etag | current attachment etag, used for concurrency checks (`null` to skip check) |

## Example

<TabItem value="delete_2" label="delete_2">
<CodeBlock language="csharp">
{`store
	.DatabaseCommands
	.DeleteAttachment("albums/holidays/sea.jpg", null);
`}
</CodeBlock>
</TabItem>


</LanguageContent>

<LanguageContent language="http">


**DeleteAttachment** is used to remove an attachment from a database.

## Syntax

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverName\}/databases/\{databaseName\}/static/\{key\} \\
	-X DELETE
`}
</CodeBlock>
</TabItem>

### Request

| Query parameter | Required |  Description |
| ------------- | -- | ---- |
| **key** | Yes | key of an attachment to delete |

| Header | Required | Description |
| --------| ------- | --- |
| **If-None-Match** | No | current attachment etag, used for concurrency checks |

### Response

| Status code | Description |
| ----------- | - |
| `204` | No content |
