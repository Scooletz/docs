# Commands: Transformers: Put
<LanguageSwitcher supportedLanguages={supportedLanguages} />
<LanguageContent language="csharp">


**PutTransformer** is used to add a transformer to a database.

## Syntax

<TabItem value="put_1" label="put_1">
<CodeBlock language="csharp">
{`string PutTransformer(string name, TransformerDefinition transformerDef);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **name** | string | name of a transformer |
| **transformerDef** | [TransformerDefinition](../../../glossary/transformer-definition.mdx) | definition of a transformer |

| Return Value | |
| ------------- | ----- |
| string | Transformer **name**. |

## Example

<TabItem value="put_2" label="put_2">
<CodeBlock language="csharp">
{`string transformerName = store
	.DatabaseCommands
	.PutTransformer(
		"Order/Statistics",
		new TransformerDefinition
		\{
			TransformResults = @"from order in results
								select new
								\{
									order.OrderedAt,
									order.Status,
									order.CustomerId,
									CustomerName = LoadDocument(order.CustomerId).Name,
									LinesCount = order.Lines.Count
								\}",
                        Name = "Order/Statistics"
		\});
`}
</CodeBlock>
</TabItem>


</LanguageContent>

<LanguageContent language="http">


**PutTransformer** is used to add a transformer to a database.

## Syntax

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverName\}/databases/\{databaseName\}/transformers/\{transformerName\} \\
	-X PUT \\
	-d @transformerDefiniton.txt
`}
</CodeBlock>
</TabItem>

### Request

| Payload |
| ------- |
| [TransformerDefinition](../../../glossary/transformer-definition.mdx) |

| Query parameter | Required | Description  |
| ------------- | -- | ---- |
| **transformerName** | Yes | name of a transformer |

### Response

| Status code | Description |
| ----------- | - |
| `201` | Created |

| Return Value | Description |
| ------------- | ------------- |
| **Transformer** | Transformer **name**. |
