**DeleteTransformer** is used to remove a transformer from a database.

## Syntax

<TabItem value="delete_1" label="delete_1">
<CodeBlock language="csharp">
{`void DeleteTransformer(string name);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **name** | string | name of a transformer to delete |

<TabItem value="delete_2" label="delete_2">
