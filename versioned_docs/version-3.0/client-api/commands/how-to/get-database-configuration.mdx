# Commands: How to get database configuration?
<LanguageSwitcher supportedLanguages={supportedLanguages} />
<LanguageContent language="csharp">


`GetDatabaseConfiguration` from admin database commands can be used to retrieve database configuration.

## Syntax

<TabItem value="database_configuration_1" label="database_configuration_1">
<CodeBlock language="csharp">
{`RavenJObject GetDatabaseConfiguration();
`}
</CodeBlock>
</TabItem>

| Return Value | |
| ------------- | ----- |
| RavenJObject | Database configuration in JSON format |

## Example

<TabItem value="database_configuration_2" label="database_configuration_2">
<CodeBlock language="csharp">
{`RavenJObject configuration = store
	.DatabaseCommands
	.Admin
	.GetDatabaseConfiguration();
`}
</CodeBlock>
</TabItem>


</LanguageContent>
<LanguageContent language="java">


`GetDatabaseConfiguration` from admin database commands can be used to retrieve database configuration.

## Syntax

<TabItem value="database_configuration_1" label="database_configuration_1">
<CodeBlock language="java">
{`RavenJObject getDatabaseConfiguration();
`}
</CodeBlock>
</TabItem>

| Return Value | |
| ------------- | ----- |
| **RavenJObject** | Database configuration in JSON format |

## Example

<TabItem value="database_configuration_2" label="database_configuration_2">
<CodeBlock language="java">
{`RavenJObject configuration = store
	.getDatabaseCommands()
		.getAdmin()
		.getDatabaseConfiguration();
`}
</CodeBlock>
</TabItem>


</LanguageContent>
<LanguageContent language="http">


`GetDatabaseConfiguration` from admin database commands can be used to retrieve database configuration.

## Syntax

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverName\}/databases/\{databaseName\}/debug/config \\
	-X GET
`}
</CodeBlock>
</TabItem>

### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |
