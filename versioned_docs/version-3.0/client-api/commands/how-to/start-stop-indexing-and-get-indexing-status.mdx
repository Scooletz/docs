## StartIndexing

This methods starts indexing, if it was previously stopped.

### Syntax

<TabItem value="start_indexing_1" label="start_indexing_1">
<CodeBlock language="csharp">
{`void StartIndexing(int? maxNumberOfParallelIndexTasks = null);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **maxNumberOfParallelIndexTasks** | int? | if set then maximum number of parallel indexing tasks will be set to this value |

### Example

<TabItem value="start_indexing_2" label="start_indexing_2">
<CodeBlock language="csharp">
{`store.DatabaseCommands.Admin.StartIndexing();
`}
</CodeBlock>
</TabItem>

## GetIndexingStatus

This methods retrieves current status of the indexing.

### Syntax

<TabItem value="get_indexing_status_1" label="get_indexing_status_1">
<CodeBlock language="csharp">
{`IndexingStatus GetIndexingStatus();
`}
</CodeBlock>
</TabItem>

| Return Value | |
| ------------- | ----- |
| string | `"Indexing"` if the indexing is running, `"Paused"` otherwise. |

### Example

<TabItem value="get_indexing_status_2" label="get_indexing_status_2">
<CodeBlock language="csharp">
{`store.DatabaseCommands.Admin.StopIndexing();
IndexingStatus status = store.DatabaseCommands.Admin.GetIndexingStatus(); // "Paused"
store.DatabaseCommands.Admin.StartIndexing();
status = store.DatabaseCommands.Admin.GetIndexingStatus(); // "Indexing"
`}
</CodeBlock>
</TabItem>

## StartIndexing

Starts indexing, if it was previously stopped.

### Syntax

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverName\}/databases/\{databaseName\}/admin/startIndexing? \\
	&concurrency=\{concurrency\} \\
	-X POST 
`}
</CodeBlock>
</TabItem>

### Request

| Query parameter | Required | Description |
| ------------- | -- | ---- |
| **concurrency** | No | if set then maximum number of parallel indexing tasks will be set to this value |

### Response

| Status code | Description |
| ----------- | - |
| `204` | No Content |

---

### Example

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl -X POST "http://localhost:8080/databases/NorthWind/admin/StartIndexing" 
< HTTP/1.1 204 No Content
`}
</CodeBlock>
</TabItem>

## GetIndexingStatus

Retrieves current status of the indexing.

### Syntax

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl -X GET http://\{serverName\}/databases/\{databaseName\}/admin/IndexingStatus
`}
</CodeBlock>
</TabItem>

### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |

| Return Value | Description |
| ------------- | ------------- |
| **IndexingStatus** | `Indexing` or `Paused` |

---

### Example

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl -X GET "http://localhost:8080/databases/NorthWind/admin/IndexingStatus" 
< HTTP/1.1 200 OK
\{"IndexingStatus":"Indexing"\}
`}
</CodeBlock>
