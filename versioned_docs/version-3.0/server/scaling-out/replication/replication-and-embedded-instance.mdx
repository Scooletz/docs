# Replication: Using Embedded instance

[Replication](../../../server/scaling-out/replication/how-replication-works.mdx) works using plain HTTP requests to replicate all changes from one server instance to another. When running in embedded mode, you can't replicate to that instance (since it has no HTTP endpoints), unless you use [Embedded+HTTP mode](../../../server/installation/embedded.mdx) but you can replicate from that instance.

## Replicating from embedded instance

To replicate from Embedded instance to another HTTP-enabled server instance you need to:

- enable `Replication` bundle on a database e.g. you can create new database with the following code:

<TabItem value="from_embedded_server_1" label="from_embedded_server_1">
<CodeBlock language="csharp">
{`store
	.DatabaseCommands
	.GlobalAdmin
	.CreateDatabase(
		new DatabaseDocument
			\{
				Id = "Northwind",
				// other configuration options omitted for simplicity
				Settings =
					\{
						\{ Constants.ActiveBundles, "Replication" \}
					\}
			\});
`}
</CodeBlock>
</TabItem>

- setup a replication by creating the `ReplicationDestinations` document with appropriate settings.

<TabItem value="from_embedded_server_2" label="from_embedded_server_2">
<CodeBlock language="csharp">
{`using (IDocumentSession session = store.OpenSession("Northwind"))
\{
	session.Store(new ReplicationDocument
	\{
		Destinations =
			\{
				new ReplicationDestination
					\{
						Url = "http://destination-server-url:8080/", 
						Database = "destination_database_name"
					\},
			\}
	\});
\}
`}
</CodeBlock>
</TabItem>

## Replicating to embedded instance

Replication requires HTTP endpoints on destination server to be running, yet, by default, when  an Embedded instance is used, internal HTTP-server is disabled. To turn it on initialize `EmbeddableDocumentStore` with `UseEmbeddedHttpServer` set to `true` (you can read more [here](../../../server/installation/embedded.mdx)).

<TabItem value="from_embedded_server_3" label="from_embedded_server_3">
<CodeBlock language="csharp">
{`EmbeddableDocumentStore store = new EmbeddableDocumentStore
	            \{
		            UseEmbeddedHttpServer = true
	            \};
`}
</CodeBlock>
</TabItem>

Now you can replicate to an embedded instance, not just from it.
