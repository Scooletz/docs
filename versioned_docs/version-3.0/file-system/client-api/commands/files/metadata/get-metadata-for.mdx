#Commands: GetMetadataForAsync

**GetMetadataForAsync** is used to retrieve the file's metadata.



<TabItem value="get_metadata_1" label="get_metadata_1">
<CodeBlock language="csharp">
{`Task<RavenJObject> GetMetadataForAsync(string filename);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **filename** | string | The name of a file |

---

| Return Value | |
| ------------- | ------------- |
| **Task&lt;RavenJObject&gt;** |  A task that represents the asynchronous metadata download operation. The task result is the file's metadata. |

#Commands: GetMetadata

The **HEAD** method is used to retrieve the file's metadata.



<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverUrl\}/fs/\{fileSystemName\}/files/\{name\}  \\
	-X HEAD 
    -I
`}
</CodeBlock>
</TabItem>

### Request

| Query parameter | Required | Description |
| ------------- | -- | ---- |
| **name** | Yes | The name of a file|


### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |
| `404` | The file was not found |

| Return Value | Description |
| ------------- | ------------- |
| **Headers** | The metadata are returned in the response's headers |
