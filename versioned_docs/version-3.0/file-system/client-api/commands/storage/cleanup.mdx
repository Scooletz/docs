#Commands: CleanUpAsync

**CleanUpAsync** forces to run a background task that will clean up files marked as deleted. Read [Background tasks](../../../server/background-tasks.mdx) article for details.



<TabItem value="clean_up_1" label="clean_up_1">
<CodeBlock language="csharp">
{`Task CleanUpAsync();
`}
</CodeBlock>
</TabItem>


| Return Value | |
| ------------- | ------------- |
| **Task** |  A task that represents the asynchronous operation |

## Example

<TabItem value="clean_up_2" label="clean_up_2">
<CodeBlock language="csharp">
{`await store.AsyncFilesCommands.Storage
		.CleanUpAsync();
`}
</CodeBlock>

#Commands: CleanUp

The **POST** method forces to run a background task that will clean up files marked as deleted. Read [Background tasks](../../../server/background-tasks.mdx) article for details.



<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverUrl\}/fs/\{fileSystemName\}/storage/cleanup  \\
	-X POST \\
    -d ""
`}
</CodeBlock>
</TabItem>

### Request

| Payload |
| ------- |
| Empty |

### Response

| Status code | Description |
| ----------- | - |
| `204` | NoContent |
