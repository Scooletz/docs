#Commands: ResetIndexes

**ResetIndexes** forces RavenFS to rebuild Lucene indexes from scratch.



<TabItem value="reset_indexes_1" label="reset_indexes_1">
<CodeBlock language="csharp">
{`Task ResetIndexes(string filesystemName);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **filesystemName** | string | The name of the file system |

---

| Return Value | |
| ------------- | ------------- |
| **Task** |  A task that represents the asynchronous restore operation |

## Example

<TabItem value="reset_indexes_2" label="reset_indexes_2">
<CodeBlock language="csharp">
{`await store.AsyncFilesCommands.Admin
	.ResetIndexes("NorthwindFS");
`}
</CodeBlock>
</TabItem>

#Commands: ResetIndexes

The **POST** method forces RavenFS to rebuild Lucene indexes from scratch.



<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverUrl\}/fs/\{filesystemName\}/admin/reset-index  \\
	-X POST \\
    -d ""
`}
</CodeBlock>
</TabItem>

### Request

| Payload |
| ------- |
| Empty |

### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |

| Return Value | Description |
| ------------- | ------------- |
| **None** | The request does not return any message |
