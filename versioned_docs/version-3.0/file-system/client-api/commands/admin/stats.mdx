#Getting file systems stats



This method returns the names of all existing file systems in the server.

### Syntax

<TabItem value="get_names_1" label="get_names_1">
<CodeBlock language="csharp">
{`Task<string[]> GetNamesAsync();
`}
</CodeBlock>
</TabItem>

| Return Value | |
| ------------- | ------------- |
| **Task&lt;string[]&gt;** | A task that represents the asynchronous restore operation. The task result is the array containing file names |

### Example

<TabItem value="get_names_2" label="get_names_2">
<CodeBlock language="csharp">
{`string[] fsNames = await store.AsyncFilesCommands.Admin
	.GetNamesAsync();
`}
</CodeBlock>
</TabItem>

`}
</CodeBlock>
</TabItem>

| Return Value | |
| ------------- | ------------- |
| **Task&lt;FileSystemStats[]&gt;** | A task that represents the asynchronous restore operation. The task result is the array containing file names |

### Example

<TabItem value="get_stats_2" label="get_stats_2">

#Getting file systems stats



The **GET** method returns the names of all existing file systems in the server.

## Syntax

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverUrl\}/fs \\
	-X GET
`}
</CodeBlock>
</TabItem>

### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |

| Return Value | Description |
| ------------- | ------------- |
| **JSON Array** | The task result is the array containing file names |

---

## Example

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	-X GET http://localhost:8080/fs
< HTTP/1.1 200 OK

[
    "NewNorthwindFS",
    "NorthwindFS"
]
`}
</CodeBlock>
</TabItem>



## Stats of all file systems

The **GET** method returns statistics of currently loaded file systems.

## Syntax

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverUrl\}/fs/stats \\
	-X GET
`}
</CodeBlock>
</TabItem>

### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |

| Return Value | Description |
| ------------- | ------------- |
| **JSON Array** | The array containing stats of loaded file systems |

## Stats of particular file system

The **GET** method returns statistics of a selected file system.

## Syntax

<TabItem value="json" label="json">
<CodeBlock language="json">
{`curl \\
	http://\{serverUrl\}/fs/\{fileSystemName\}/stats \\
	-X GET
`}
</CodeBlock>
</TabItem>

### Response

| Status code | Description |
| ----------- | - |
| `200` | OK |

| Return Value | Description |
| ------------- | ------------- |
| **JSON Array** | The file system stats |
