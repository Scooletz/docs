# Query With Exact Match
<LanguageSwitcher supportedLanguages={supportedLanguages} />
<LanguageContent language="csharp">


By default, the `Where` method in `Query` uses a case-insensitive match.

To perform a case-sensitive match you should use the `exact` parameter.

### Syntax

<TabItem value="query_1_0" label="query_1_0">
<CodeBlock language="csharp">
{`IRavenQueryable<T> Where<T>(Expression<Func<T, int, bool>> predicate, bool exact);

IRavenQueryable<T> Where<T>(Expression<Func<T, bool>> predicate, bool exact);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **predicate** | Expression&lt;Func&lt;T, int, bool&gt;&gt; | Predicate with match condition |
| **exact** | bool | Indicates if `predicate` should be matched in case-sensitive manner |

### Example I - Query With Exact Match

<Tabs groupId='languageSyntax'>
<TabItem value="Sync" label="Sync">
<CodeBlock language="csharp">
{`// load all entities from 'Employees' collection
// where FirstName equals 'Robert' (case sensitive match)
List<Employee> employees = session
    .Query<Employee>()
    .Where(x => x.FirstName == "Robert", exact: true)
    .ToList();
`}
</CodeBlock>
</TabItem>
<TabItem value="Async" label="Async">
<CodeBlock language="csharp">
{`// load all entities from 'Employees' collection
// where FirstName equals 'Robert' (case sensitive match)
List<Employee> employees = await asyncSession
    .Query<Employee>()
    .Where(x => x.FirstName == "Robert", exact: true)
    .ToListAsync();
`}
</CodeBlock>
</TabItem>
<TabItem value="RQL" label="RQL">
<CodeBlock language="sql">
{`from Employees where exact(FirstName == 'Robert')
`}
</CodeBlock>
</TabItem>
</Tabs>

### Example II - Query With Inner Exact Match

<Tabs groupId='languageSyntax'>
<TabItem value="Sync" label="Sync">
<CodeBlock language="csharp">

    .ToList();
`}
</CodeBlock>
</TabItem>
<TabItem value="Async" label="Async">
<CodeBlock language="csharp">
{`// return all entities from 'Orders' collection
// which contain at least one order line with

    .Query<Order>()
    .Where(x => x.Lines.Any(p => p.ProductName == "Singaporean Hokkien Fried Mee"), exact: true)

`}
</CodeBlock>
</TabItem>

</Tabs>

<LanguageContent language="java">


By default, the `whereXXX` methods in `query` uses a case-insensitive match.

To perform a case-sensitive match you should use the `exact` parameter.

### Syntax

<TabItem value="query_1_0" label="query_1_0">
<CodeBlock language="java">
{`IDocumentQuery<T> whereEquals(String fieldName, Object value, boolean exact);
IDocumentQuery<T> whereNotEquals(String fieldName, Object value, boolean exact);

// ... rest of where methods
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **fieldName** | String | Object field to use |
| **value** | Object | Predicate value |
| **exact** | boolean | Indicates if `predicate` should be matched in case-sensitive manner |

### Example I - Query With Exact Match

<Tabs groupId='languageSyntax'>

`}
</CodeBlock>
</TabItem>

<LanguageContent language="nodejs">


By default, the `whereXXX()` methods in `query` use a case-insensitive match.

To perform a case-sensitive match you should use the `exact` parameter.

### Syntax

<TabItem value="query_1_0" label="query_1_0">
<CodeBlock language="js">
{`query.whereEquals(fieldName, value, [exact]);
query.whereNotEquals(fieldName, value, [exact]);

// ... rest of where methods
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ------------- | ------------- | ----- |
| **fieldName** | string | Object field to use |
| **value** | any | Predicate value |
| **exact** | boolean | Indicates if `predicate` should be matched in case-sensitive manner |

### Example I - Query With Exact Match

<Tabs groupId='languageSyntax'>
<TabItem value="Node.js" label="Node.js">
<CodeBlock language="js">
{`// load all entities from 'Employees' collection
// where firstName equals 'Robert' (case sensitive match)
const employees = await session.query({ collection: "Employees" })
    .whereEquals("FirstName", "Robert", true)
    .all();
`}
</CodeBlock>
</TabItem>
<TabItem value="RQL" label="RQL">
<CodeBlock language="sql">

</CodeBlock>
</TabItem>
</Tabs>
