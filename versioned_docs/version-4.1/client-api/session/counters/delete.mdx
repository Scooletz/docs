# Deleting a Counter  
<LanguageSwitcher supportedLanguages={supportedLanguages} />
<LanguageContent language="csharp">

<Admonition type="note" title="">

* All the Counters for a document are deleted when the document is deleted.  

* You can also use the [CountersFor](../../../client-api/session/counters/overview.mdx#counter-methods-and-the--object).`Delete` method to remove a specific Counter from a document.

    - [`Delete ` Syntax](../../../client-api/session/counters/delete.mdx#delete-syntax)

</Admonition>
## `Delete ` Syntax

<TabItem value="Delete-definition" label="Delete-definition">
<CodeBlock language="csharp">
{`void Delete(string counterName);
`}
</CodeBlock>
</TabItem>

| Parameter | Type | Description |
|:-------------:|:-------------:|:-------------:|
| `counterName` |  string | Counter's name |


## `Delete ` Usage

*  **Flow**:  
  * Open a session  
  * Create an instance of `CountersFor`.  
      * Either pass `CountersFor` an explicit document ID, -or-  
      * Pass it an [entity tracked by the session](../../../client-api/session/loading-entities.mdx), e.g. a document object returned from [session.query](../../../client-api/session/querying/how-to-query.mdx) or from [session.Load](../../../client-api/session/loading-entities.mdx#load).  
  * Execute `CountersFor.Delete`
  * Execute `session.SaveChanges` for the changes to take effect  

* **Note**:
    * A Counter you deleted will be removed only after the execution of `SaveChanges()`.  
    * Deleting a document deletes its Counters as well.  
    * `Delete` will **not** generate an error if the Counter doesn't exist.  



## Code Sample

<TabItem value="counters_region_Delete" label="counters_region_Delete">
<CodeBlock language="csharp">
{`// 1. Open a session
using (var session = docStore.OpenSession())
\{
    // 2. pass CountersFor's constructor a document ID  
    var documentCounters = session.CountersFor("products/1-C");

    // 3. Delete the "ProductLikes" Counter
    documentCounters.Delete("ProductLikes");

    // 4. Save changes to the session
    session.SaveChanges();
\}
`}
</CodeBlock>
</TabItem>
