The **GetTermsOperation** will retrieve stored terms for a field of an index.

## Syntax

<TabItem value="get_terms1" label="get_terms1">
<CodeBlock language="csharp">
{`public GetTermsOperation(string indexName, string field, string fromValue, int? pageSize = null)
`}
</CodeBlock>
</TabItem>


| Parameters | | |
| ------------- | ------------- | ----- |
| **indexName** | string | Name of an index to get terms for |
| **field** | string | Name of field to get terms for |
| **fromValue** | string | Starting point for a query, used for paging |
| **pageSize** | int? | Maximum number of terms that will be returned |

| Return Value | |
| ------------- | ----- |
| string[] | Array of index terms. |

## Example

<TabItem value="get_terms2" label="get_terms2">
<CodeBlock language="csharp">
{`string[] terms = store.Maintenance.Send(new GetTermsOperation("Orders/Totals", "Employee", null));
`}
</CodeBlock>
</TabItem>
