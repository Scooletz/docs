# Bundle: Compression

When creating a database, if you want to use the compression bundle you need to select it in the database creation window:  
![Tasks Fig 1](./assets/studio_compression_1.PNG)  

After pressing "Next" your database will be created and compression enabled.

<Admonition type="note" title="">
Activating compression bundle is only supported for new databases. Activating or disabling compression on an already existing database will cause DB malfunction.
</Admonition>

<Admonition type="info" title="">
You can read more about the Compression Bundle in [here](../../server/extending/bundles/compression.mdx).
</Admonition>
