# Data Archival
<Admonition type="note" title="Note">

* As a database grows, basic functions like indexing can slow down.  
  To address this, RavenDB offers the ability to **archive selected documents**.  
 
* Documents that are rarely accessed or no longer relevant for active use,  
  but still need to be kept for compliance or historical purposes, can be archived.

* This feature is covered in detail in the following articles:
  * [Data archival overview](../../../data-archival/overview.mdx)
  * [Enable data archiving](../../../data-archival/enable-data-archiving.mdx)
  * [Schedule document archiving](../../../data-archival/schedule-document-archiving.mdx)
  * [Archived documents and other features](../../../data-archival/archived-documents-and-other-features.mdx)
  * [Unarchiving documents](../../../data-archival/unarchiving-documents.mdx)

</Admonition>
