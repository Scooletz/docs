## System Requirements

RavenDB comes in several flavors. Listed below are the system requirements for each.

### Lightweight Client

* .NET Framework 4.0 Client Profile
* The client doesn't require Windows and **can run on Mono**.

### Silverlight Client

* Silverlight 4 or 5

### RavenDB Server / Embedded

* .NET Framework 4.0
* Windows
