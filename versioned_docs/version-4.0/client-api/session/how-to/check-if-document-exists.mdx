# Session: How to Check if a Document Exists
<LanguageSwitcher supportedLanguages={supportedLanguages} />
<LanguageContent language="csharp">


In order to check if a document with specific ID exists in the database, use the `Exists` method from the `Advanced` session operations.

## Syntax

<TabItem value="exists_1" label="exists_1">
<CodeBlock language="csharp">
{`bool Exists(string id);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ---------- | ---------- | ----- |
| **id** | string | ID of the document to check the existence of. |

| Return Value | |
| ------------- | ----- |
| bool | Indicates if a document with the given ID exists. |

## Example

<TabItem value="exists_2" label="exists_2">
<CodeBlock language="csharp">
{`bool exists = session.Advanced.Exists("employees/1-A");

if (exists)
\{
    //do something
\}
`}
</CodeBlock>
</TabItem>


</LanguageContent>

<LanguageContent language="nodejs">


In order to check if a document with specific ID exists in the database, use the `exists()` method from the `advanced` session operations.

## Syntax

<TabItem value="exists_1" label="exists_1">
<CodeBlock language="js">
{`session.advanced.exists(id);
`}
</CodeBlock>
</TabItem>

| Parameters | | |
| ---------- | ---------- | ----- |
| **id** | string | ID of the document to check the existence of. |

| Return Value | |
| ------------- | ----- |
| `Promise<boolean>` | Indicates if a document with the given ID exists. |

## Example

<TabItem value="exists_2" label="exists_2">

\}
`}
