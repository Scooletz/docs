# Migration: How to Migrate MoreLikeThis from 3.x

MoreLikeThis functionality has been merged into [RQL](../../../../indexes/querying/what-is-rql.mdx). To reflect this change the Client API has integrated this feature into the `session.Query` and `session.Advanced.DocumentQuery`. The following migration samples will focus on the `session.Query` - the most common and recommended way of interaction with querying capabilities on RavenDB.

## Example I

| 3.x |
|:---:|
|
<TabItem value="more_like_this_1" label="more_like_this_1">
<CodeBlock language="csharp">
{`Article[] articles = session
    .Advanced
    .MoreLikeThis<Article>(
        "Articles/MoreLikeThis",
        null,
        new MoreLikeThisQuery
        \{
            IndexName = "Articles/MoreLikeThis",
            DocumentId = "articles/1",
            Fields = new[] \{ "Body" \}
        \});
`}
</CodeBlock>
</TabItem>
|

| 4.0 |
|:---:|
|
<TabItem value="more_like_this_2" label="more_like_this_2">
<CodeBlock language="csharp">
{`List<Article> articles = session
    .Query<Article>("Articles/MoreLikeThis")
    .MoreLikeThis(builder => builder
        .UsingDocument(x => x.Id == "articles/1")
        .WithOptions(new MoreLikeThisOptions
        \{
            Fields = new[] \{ "Body" \}
        \}))
    .ToList();
`}
</CodeBlock>
</TabItem>
|
