# Migration: DatabaseCommands has been removed

`IDatabaseCommands` and `IAsyncDatabaseCommands` has been replaced in RavenDB.Client 4.0 by operations. Please read the dedicated documentation section about [operations](../../client-api/operations/what-are-operations.mdx) available in the client API.
