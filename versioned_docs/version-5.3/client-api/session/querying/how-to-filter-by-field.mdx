# Filter by Field Presence  
<LanguageSwitcher supportedLanguages={supportedLanguages} />
<LanguageContent language="csharp">

<Admonition type="note" title="">

* To query for documents that contain a particular field,  
  use extension method `WhereExists()` which is accessible from [DocumentQuery](../../../client-api/session/querying/document-query/what-is-document-query.mdx).  

* A document that doesn't contain the specified field will be excluded from the query results.

* In this page:  
  * [Filter by field name](../../../client-api/session/querying/how-to-filter-by-field.mdx#filter-by-field-name)  
  * [Filter by field path](../../../client-api/session/querying/how-to-filter-by-field.mdx#filter-by-field-path)  
  * [Syntax](../../../client-api/session/querying/how-to-filter-by-field.mdx#syntax)

</Admonition>
## Filter by field name 

<Tabs groupId='languageSyntax'>
<TabItem value="Sync" label="Sync">
<CodeBlock language="csharp">
{`// Only documents that contain field 'FirstName' will be returned

List<Employee> results = session
    .Advanced
    .DocumentQuery<Employee>()
    .WhereExists("FirstName")
     // Or use lambda expression: .WhereExists(x => x.FirstName)
    .ToList();
`}
</CodeBlock>
</TabItem>
<TabItem value="Async" label="Async">
<CodeBlock language="csharp">
{`// Only documents that contain field 'FirstName' will be returned

List<Employee> results = await asyncSession
    .Advanced
    .AsyncDocumentQuery<Employee>()
    .WhereExists("FirstName")
     // Or use lambda expression: .WhereExists(x => x.FirstName)
    .ToListAsync();
`}
</CodeBlock>
</TabItem>
<TabItem value="RQL" label="RQL">
<CodeBlock language="sql">
{`// Only documents that contain field 'FirstName' will be returned

from Employees
where exists("FirstName")
`}
</CodeBlock>
</TabItem>
</Tabs>

## Syntax

<TabItem value="whereExists_syntax" label="whereExists_syntax">
<CodeBlock language="csharp">
{`IDocumentQuery<T> WhereExists(string fieldName);

IDocumentQuery<T> WhereExists<TValue>(Expression<Func<T, TValue>> propertySelector);
`}
</CodeBlock>
</TabItem>

| Parameters           | Type                         | Description                                                           |
|----------------------|------------------------------|-----------------------------------------------------------------------|
| **fieldName**        | `string`                     | The name / path of the document field to filter by                    |
| **propertySelector** | `Expression<Func<T,TValue>>` | Lambda expression with name / path of the document field to filter by |




</LanguageContent>
<LanguageContent language="java">

<Admonition type="note" title="">

* To filter documents by whether they contain a particular field, use the LINQ extension method `whereExists()` accessible 
from the [Document Query](../../../client-api/session/querying/document-query/what-is-document-query.mdx).  

* If a document doesn't contain the specified field, it is excluded from the query results.

* In this page:  
  * [Syntax](../../../client-api/session/querying/how-to-filter-by-field.mdx#syntax)  
  * [Examples](../../../client-api/session/querying/how-to-filter-by-field.mdx#examples)  

</Admonition>
## Syntax

<TabItem value="whereexists_1" label="whereexists_1">
<CodeBlock language="java">
{`IDocumentQuery<T> whereExists(String fieldName);
`}
</CodeBlock>
</TabItem>

| Parameters | Type | Description |
| - | - | - |
| **fieldName** | `String` | The name or path to the field you want to filter by |
