# Cloud Portal: The API Tab

<Admonition type="note" title="Note">

The RavenDB Cloud API allows you to automate actions on your RavenDB Cloud products. By making simple HTTP calls, you can adjust your storage size or change the instance type. Scale up or down without visiting the RavenDB Cloud portal.

## The API Tab

!["Figure 1 - The API Tab"](./assets/portal-api-tab-overview.png)

You can read more about the *RavenDB Cloud API* in the [API](../../cloud/cloud-api) page.
### Create API Key

!["Figure 2 - Create API Key"](./assets/portal-api-tab-empty-api-key-list.png)

**1**. Click the *Add API Key* button.  

!["Figure 3 - Create API Key - configuration"](./assets/portal-api-tab-create-api-key-modal.png)

**2**. Enter an API Key description.  
**3**. Select an API Key expiration date.  
**4**. Click the *Create* button.

!["Figure 4 - Create API Key - API Key"](./assets/portal-api-tab-create-api-key-modal-copy-api-key.png)

**5**. Copy your API Key.  
**6**. Click the *Close* button.
### Edit API Key

!["Figure 5 - Edit API Key"](./assets/portal-api-tab-with-api-keys-edit-api-key.png)

**1**. Click the *Edit* button.

!["Figure 6 - Edit API Key - configuration"](./assets/portal-api-tab-edit-api-key-modal.png)

**2**. Enter an API Key description.  
**3**. Select an API Key expiration date.  
**4**. Click the *Save* button.
### Deactivate API Key

!["Figure 7 - Deactivate API Key"](./assets/portal-api-tab-with-api-keys-deactivate-api-key.png)

**1**. Click the *Deactivate* button.

!["Figure 8 - Deactivate API Key - confirmation"](./assets/portal-api-tab-deactivate-api-key-modal.png)

**2**. Click the *Deactivate* button.
